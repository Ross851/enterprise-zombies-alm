The Two-Minute Test

If you can't identify who owns an application and locate its core documentation within two minutes, you're already dealing with a zombie.

A simple but powerful tool for identifying zombie applications.

  1. Step 1: Ask “Who is responsible for this application?”
  2. Step 2: Ask “Where is the documentation for this application?”
  3. Step 3: Set a timer for two minutes.
  4. Step 4: If either question remains unanswered when time expires, you've identified a zombie.
“The power of the Two-Minute Test lies in its simplicity—if no one knows, no one owns it.”
